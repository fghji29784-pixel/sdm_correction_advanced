# SDM 보정법 비교 (sdm_correction_advanced)

리튬이차전지 자가방전 단시간 판정에서 **측정 전류 보정법**을 비교·평가하는 **독립 실행** 도구.
`보정법_후보_조사.md`의 종합 권장안(A-1 / C-1 / D-2 / D-1)을 단계적으로 적용해
어떤 보정 조합이 가장 우수한지 한 번에 비교한다.

> **독립 실행**: `sdm_logic` 프로젝트가 없어도 단독으로 동작한다.
> 원본 측정 파일(또는 이미 파싱된 결과 CSV)만 넣으면 보정법 비교표가 출력된다.

---

## 1. 설치

```bash
pip install -r requirements.txt
```

(Anaconda 환경이면 numpy/pandas/statsmodels/sklearn은 대개 이미 설치돼 있음)

---

## 2. 실행 방법 (3가지 입력)

### (1) 원본 측정 폴더 + 공정 데이터
KSS 파일들과 TEMP 파일이 든 트레이 폴더(들)의 **상위 폴더**를 지정.

```bash
python main.py --data "C:\측정\상위폴더" --process "공정데이터.xlsx"
```

상위 폴더 구조 예:
```
상위폴더/
├── CFDD000001/   ← 트레이 1 (KSS 1~5개 + TEMP 파일)
├── CFDD000002/   ← 트레이 2
└── ...
```

### (2) 이미 파싱된 df_meta CSV (sdm_logic 결과 테이블)
`sdm_logic` 툴에서 내보낸 통합 결과 테이블을 그대로 사용.

```bash
python main.py --meta-csv "df_meta.csv"
```
> 필수 컬럼: `cell_no`, `i_15min`(또는 `i_Nmin`), `t_final`, `rwiring`, `tray_id`
> 선택 컬럼: `판정등급`(d_prime용), `Delta OCV_Delta OCV #07`(dOCV 검증용)

### (3) 데모 (데이터 없이 동작 확인)
```bash
python main.py --demo
```

## 5. 비교되는 보정 구성

| 구성 | 적용 보정 | 분류 |
|---|---|---|
| `[0] Baseline` | 기존 선형 회귀 (v_init·온도·레이어 더미) | 기준선 |
| `[1] +A1` | **A-1** Arrhenius 곱셈형 온도 보정 | 물리 ★★★ |
| `[2] +A1+C1` | + **C-1** Rwiring→τ 채널별 보정 | 물리 ★★★ |
| `[3] +A1+C1+D2` | + **D-2** 공간 추세면(2차 다항, 레이어 더미 대체) | 통계 ★★★ |
| `[4] +A1+C1+D2+D1` | + **D-1** 혼합효과 모델(트레이 랜덤절편) | 통계 ★★★ |

### 보정법 요약
- **A-1 Arrhenius** — `I_corr = I·exp[(Eact/k_B)(1/T_meas − 1/T_ref)]`. 자가방전의 지수적 온도 의존(Eact=0.94eV)을 물리식으로 환산. 회귀 계수 추정 불필요 → 불량 0개 트레이에서도 작동.
- **C-1 Rwiring→τ** — `I_corr = I·(Rout+Rser+Rwiring_ch)/(Rout+Rser+Rwiring_ref)`. 접촉저항 편차로 인한 τ 차이 보정. 채널 제거 대신 **보정**으로 전환 → 데이터 손실 없음.
- **D-2 공간 추세면** — 레이어 이산 더미 5개를 `row, col, row², col², row·col` 연속 다항으로 대체. 중앙 고온/가장자리 저온 구배를 매끄럽게 포착.
- **D-1 혼합효과** — 트레이를 랜덤절편으로 두어 공통 기울기는 공유(부분 풀링), 트레이 레벨차만 흡수. 소표본 트레이 안정화.

## 6. 평가 지표 해석

| 지표 | 의미 |
|---|---|
| **d_prime** | 불량/양품 분리도. 불량 라벨(E) 있을 때만 유효. 소수 불량 의존 → 보조 지표. |
| **dOCV_r 양품만** | SDM_corrected ↔ dOCV 연속 상관(양품 셀만). **CLAUDE.md v3 핵심 지표.** 양품 범위 안에서도 dOCV를 추종하면 → 미세불량 선별 가능한 *진짜 대체재*. |

> ⚠ **주의**: 현재 보유 데이터는 명백한 단락(z≈33)만 있어 보정법 우열 판가름 불가.
> 경계성(미세) 저전압 불량 데이터 확보 후 `dOCV_r 양품만` 으로 최종 선택할 것.
