# SDM 보정 로직 고도화 — 보정법 후보 조사

> 작성: 2026-06-24
> 목적: 현재 회귀 기반 선형 보정을 고도화할 수 있는 **온도 / 전압 / 자가방전전류(ISD)** 보정법 후보를 물리·통계 양 측면에서 정리.
> 근거: 보유 논문 2편(R-Smith 2020/2023) + 외부 문헌 + 현재 `analysis.py` 구조.

## 0. 현재 보정 구조 요약 (출발점)

`analysis.py`의 보정은 **단일 다중선형회귀의 부분잔차 제거** 방식이다.

```
I_corrected = I_measured − Σ β_k · (x_k − x_k,ref)
   x_k ∈ { v_init, t_final, delta_t, dummy_L2~L6, (OCV1~4,7, 충전종료전압) }
```

특징과 한계:

| 항목 | 현재 방식 | 한계 |
|---|---|---|
| 온도 | `t_final`, `delta_t` **선형항** | ISD–T 관계는 본질적으로 **지수(Arrhenius)** → 선형 근사는 좁은 구간에서만 타당 |
| 전압 | `v_init` 선형항 | OCV 매칭 오차의 물리량(TCV·ΔT)과 분리 안 됨 |
| 위치 | 6레이어 **이산 더미** | 실제 열분포는 중앙-고온/가장자리-저온의 **연속 구배** → 이산화로 정보 손실 |
| τ/Rwiring | 보정 없음(이상치 제거만) | τ=(Rout+Rser+Rwire)·Ceff 의 채널별 차이가 선형구간 신호 크기를 직접 좌우 |
| 모델 | 트레이별 독립 OLS/Robust/Lasso | 불량 0개 트레이 다수 → 트레이마다 계수 추정 불안정 가능 |

아래 후보들은 이 표의 각 한계를 겨냥한다. 모든 후보에 **현 코드 적용 난이도**와 **권장도**를 표기했다.

## A. 온도 보정 후보

### A-1. Arrhenius 곱셈형 ISD 온도 보정 ★★★ (강력 권장)

**원리.** 자가방전 전류는 화학 부반응(Eact=0.94 eV) 지배이므로 온도와 **지수 관계**다. 논문 실측: 23°C에서 3–6 µA → 40°C에서 97 µA (약 20배). 선형항으로는 재현 불가.

```
ISD(T_ref) = ISD(T) · exp[ (Eact/R) · (1/T_ref − 1/T) ]       (T는 절대온도 K)
```

**적용 방법.** 현재 `correct_values`의 `t_final` 선형 빼기를 **곱셈 보정**으로 교체:

```python
k = (Eact_eV * 1.602e-19) / 1.380649e-23     # Eact/R  (단위 K)
I_corr = I_meas * np.exp(k * (1/T_ref_K - 1/T_meas_K))
```

**장점.** 물리적으로 정확. 트레이 내 ~0.5°C 편차뿐이라도, 곱셈형은 모든 셀을 동일 기준온도(25°C)로 환산해 트레이 간 레벨차도 줄임. 회귀로 계수 추정할 필요 없이 **Eact만 알면 적용** → 불량 0개 트레이에서도 작동.

**단점.** Eact를 0.94 eV로 고정하면 셀/공정마다 실제값이 다를 수 있음(±0.14 eV 불확실성 → 보정량 수% 변동). → 실측 데이터로 Eact 캘리브레이션 권장(여러 온도 SDM 데이터 필요).

**적용 난이도.** 낮음. `correct_values`에 곱셈 분기 추가 + `T_init/T_final` 컬럼(이미 존재) 활용.

---

### A-2. TCV(엔트로피 계수) 기반 OCV-온도 보정 ★★★ (권장)

**원리 (이번 조사 핵심).** 프로젝트의 TCV(µV/K)는 외부 문헌상 셀의 **엔트로피 계수 dOCV/dT** 와 동일 개념이다. 즉 온도 1°C 변동 → OCV가 TCV만큼 이동 → 포텐시오스태틱 매칭 전압과 셀 OCV 사이에 가짜 오프셋 → **측정 전류에 직접 오차**.

```
ΔV_thermal = TCV(SOC) · ΔT
I_offset_보정 ≈ ΔV_thermal / (Rout + Rser)      # 가짜 OCV 오프셋이 만든 전류 성분 제거
```

**적용 방법.** ① 측정 SOC(≈30%)에서의 TCV를 상수로 두거나 SOC 함수로 룩업, ② `delta_t`(또는 트레이 내 온도편차)에 TCV를 곱해 전류 오차항을 **물리식으로 직접 계산**해 빼기. 회귀의 `delta_t` 계수를 이 물리값으로 대체/검증.

**장점.** 현재 `delta_t` 선형항이 실제로 무엇을 보정하는지 물리적으로 규명. SOC 30%(TCV≈0)에서 측정 중이라면 이 항이 **0에 가까워야 정상** → 회귀계수가 크게 나오면 SOC 이탈/측정 이상 진단 신호로도 활용.

**단점.** 채널별 SOC 미세 차이와 TCV의 SOC 민감도를 알아야 정밀해짐. Rser 추정 필요.

**적용 난이도.** 중간. TCV–SOC 룩업 테이블(논문 Fig.4b 값 디지털화) + Rser 가정 필요.

---

### A-3. 트레이 고정효과 기반 온도 디컨파운딩 ★★ (보조)

**원리.** CLAUDE.md v3에서 확인된 사실: 전체 합산 `t_final` r=−0.41은 **트레이 간 교란(confounding)**이지 진짜 온도효과가 아님. 풀링 회귀에 트레이 더미(고정효과)를 안 넣으면 가짜 상관을 학습.

**적용 방법.** 풀링 모델을 쓸 경우 `C(tray_id)` 더미를 항상 포함(within-tray 효과만 추정). 현재처럼 트레이별 독립 회귀를 쓰면 이미 자동 충족 → 이 후보는 **A-1/A-2를 풀링으로 통합할 때만** 필요.

**적용 난이도.** 낮음(이미 부분 충족).

## B. 전압 / OCV 보정 후보

### B-1. dI/dV 민감도 기반 V_init 보정 ★★ (권장)

**원리.** 현재 `v_init` 선형항은 통계적으로 적합되지만, 물리적으로 V_init 편차가 전류에 미치는 영향은 **셀의 미분 용량/임피던스**에 의존. 선형구간에서 I(t)≈ISD·t/τ, τ∝Ceff. 매칭 전압 오차 δV → 초기 과도전류 ∝ δV/(Rout+Rser).

**적용 방법.** `v_init` 단순 빼기 대신 `(v_init − V_ref)/(Rout+Rser)` 형태의 물리 스케일로 보정하거나, 회귀 독립변수를 이 변환값으로 교체해 해석력↑.

**장점.** A-2와 일관된 물리 프레임. V_init·TCV·Rwiring 보정이 모두 같은 (Rout+Rser) 분모를 공유 → 통합 보정식으로 정리 가능.

**단점.** Rser·Rout 값 필요(KSS 파일의 Rout, Rser는 추정).

**적용 난이도.** 중간.

---

### B-2. Ceff / 용량 정규화 ★★ (권장, 선형구간 핵심)

**원리.** 선형구간 핵심식 `I(t) ≈ ISD·(t/τ)`, `τ=(Rout+Rser+Rwire)·Ceff`. 셀마다 Ceff(≈용량·SOC)가 다르면 **같은 ISD라도 측정 전류가 달라짐**. 공정 OCV들은 사실상 Ceff/Rser 개인차의 대리변수(CLAUDE.md 명시).

**적용 방법.** 측정 전류를 **τ로 나눠 ISD 스케일로 환산**: `ISD_proxy = I(t) · τ / t`. τ는 B-3(Rwiring)·Ceff 추정으로 산출. 종속변수를 I 대신 ISD_proxy로 두면 모든 후속 보정이 단순화.

**장점.** v2의 slope(=ISD/τ) 아이디어를 한 단계 더 밀어 **ISD 자체로 정규화** → 트레이 간/셀 간 비교가 물리적으로 정당.

**단점.** Ceff 절대값 추정이 어렵움 → 상대 τ만으로도 부분 효과.

**적용 난이도.** 중간~높음.

---

## C. 자가방전전류(ISD) / 시간상수 보정 후보

### C-1. Rwiring → τ 보정 (채널별 시간상수 정규화) ★★★ (강력 권장)

**원리.** 논문 핵심 기여 중 하나가 **Rwire in-situ 캘리브레이션**(raw→corrected 시 전류가 더 낮은 τ로 이동, Fig.1b). CLAUDE.md도 "특정 채널 Rwiring 이상→τ 증가→같은 시간에 더 낮은 전류→단시간 판정 오차"를 문제로 적시. 현재는 임계 초과 채널을 **제거만** 하고 정상 범위 편차는 **미보정**.

**적용 방법.** 선형구간에서 `I_meas(t) = ISD · t/τ` 이므로 채널별 τ를 알면:
```
ISD_proxy = I_meas(t) · τ_ch / t          # τ_ch = (Rout + Rser + Rwiring_ch)·Ceff
보정(상대): I_corr = I_meas · (τ_ch / τ_ref)
```
Rwiring은 KSS `TestSetupPerChan`에 이미 채널별로 있음 → 즉시 활용 가능.

**장점.** 제거 대신 **보정**으로 전환 → 데이터 손실 없음. 접촉불량 채널의 체계적 편향 제거. 단시간 판정 정확도 직접 개선.

**단점.** Rout·Rser·Ceff 가정 필요(상대 보정이면 Rout은 설정값으로 알 수 있음).

**적용 난이도.** 중간. Rwiring 컬럼 존재 → 보정식만 추가.

---

### C-2. 지수 피팅 외삽 (ISD 직접 추정) ★ (참고 / 장시간만)

**원리.** 논문은 `IMeas = ISD + A·exp(−t/τ)` 전체 시계열 피팅 또는 마지막 100점 평균으로 ISD 추출.

**적용 방법/한계.** CLAUDE.md가 이미 결론낸 대로, **15분(t/τ≈0.04) 선형구간에선 외삽 신뢰 불가**(수렴의 4%만 관찰). → 단시간 판정엔 부적합. 단, 일부 셀을 장시간 측정해 **τ·Ceff 보정 상수를 캘리브레이션**하는 용도로는 유효.

**적용 난이도.** 낮음(피팅) — 단 단시간 판정엔 권장 안 함.

---

### C-3. 종속변수를 slope(0~N분)로 (v2 채택안) ★★ (이미 검토됨)

**원리.** `slope ≈ ISD/τ`, N점 평균으로 순간노이즈 억제. CLAUDE.md v2에 정리됨. C-1(τ보정)과 결합하면 `slope·τ ≈ ISD`로 물리량 직결.

**적용 난이도.** 낮음(이미 `slope_0_N` 파생 존재).

## D. 통계 / 모델 기반 보정 후보

### D-1. 혼합효과(계층) 모델 — 트레이 랜덤절편 ★★★ (권장)

**원리.** 현재 "트레이별 완전 독립 회귀"는 트레이마다 계수를 새로 추정 → 셀 수 적거나 불량 0개일 때 불안정. **혼합효과모델**은 트레이를 랜덤효과로 두어 공통 기울기(온도·전압 계수)는 전 트레이에서 공유(pooling)하고, 트레이 레벨차만 랜덤절편으로 흡수.

```
I_ij = (β_공통 · x_ij) + u_tray[j] + ε_ij ,   u_tray ~ N(0, σ_tray²)
```

**적용 방법.** `statsmodels.MixedLM` 또는 `MixedLMResults`. 독립변수는 현행 유지, `groups=tray_id`.

**장점.** 부분 풀링(shrinkage)으로 소표본 트레이 안정화. 트레이 간 이봉(bimodal) 레벨차를 랜덤절편이 정확히 처리(현재 median 중심화와 같은 목적을 모델 내부에서 통계적으로 수행). 물리 보정(A·B·C)과 **상보적** — 물리 보정으로 못 잡은 잔차를 통계적으로 정리.

**단점.** 해석·수렴이 OLS보다 까다로움. 비숙련자 배포 툴에선 내부 처리로 숨겨야.

**적용 난이도.** 중간. `analysis.py`에 옵션 6으로 추가 가능.

---

### D-2. 공간 추세면 / GP 위치 보정 (이산 더미 대체) ★★★ (권장)

**원리.** 온도 히트맵은 중앙-고온/가장자리-저온의 **매끄러운 2D 구배**. 현재 6레이어 이산 더미는 이를 계단함수로 근사 → 정보 손실·경계 인공물. 셀의 (row, col) 좌표를 직접 쓰는 **2D 추세면(다항)** 또는 **가우시안 프로세스(kriging)**가 연속 구배를 자연스럽게 포착.

**적용 방법.**
- 간단판: `row, col, row², col², row·col` 다항항을 더미 대신 투입.
- 고급판: `sklearn.gaussian_process`로 위치 잔차면 추정 후 빼기(셀별 공간 보정값).

**장점.** 레이어 더미 5개 → 다항 5~6개로 비슷한 비용에 연속 보정. 측정된 온도 히트맵과 직접 대조해 검증 가능. 위치 보정이 곧 **온도 구배 보정**의 대리이므로 A-2와 시너지.

**단점.** GP는 과적합/하이퍼파라미터 주의(144점이면 다항이 더 안전).

**적용 난이도.** 낮음(다항) ~ 중간(GP). row/col은 cell_no에서 즉시 계산.

---

### D-3. 강건 회귀 고도화 — Huber / 분위수(Quantile) 회귀 ★★ (보조)

**원리.** 현행 Robust(잔차 10~90% 절단 후 WLS)는 휴리스틱. 표준적 **Huber-T M-추정** 또는 **분위수회귀(중앙값 회귀)**가 이상치(불량셀)에 원리적으로 강건하며, 불량을 자동 다운웨이팅하면서 정상셀 기준선 오염을 막음(CLAUDE.md 표준화 의도와 동일).

**적용 방법.** `statsmodels.RLM(M=sm.robust.norms.HuberT())` 또는 `QuantReg(q=0.5)`.

**장점.** 절단 백분위 하이퍼파라미터 제거. 통계적으로 표준·검증된 방법.

**단점.** 불량이 0개면 OLS와 거의 동일(현 데이터 다수가 그러함) → 효과는 불량 포함 트레이에서만.

**적용 난이도.** 낮음(옵션 교체).

---

### D-4. 엘라스틱넷 / 정규화 (Lasso 확장) ★ (이미 부분 구현)

**원리.** 공정 OCV들의 다중공선성 → Lasso(옵션5)로 변수 선택 중. **ElasticNet**은 상관 높은 OCV 그룹을 함께 살리거나 죽여 Lasso의 불안정성 보완.

**적용 난이도.** 낮음(`ElasticNetCV` 교체).

### 권장 1차 도입 묶음 (물리+통계 균형)

물리 보정 2개 + 통계 보정 2개를 **상보적으로** 결합하는 것을 권장한다:

1. **A-1 (Arrhenius)** + **C-1 (Rwiring→τ)** 로 측정 전류를 먼저 **물리적으로 ISD 스케일에 가깝게 정규화** →
2. 남은 잔차를 **D-2 (공간 추세면)** 로 위치 구배 제거, **D-1 (혼합효과)** 로 트레이 레벨차 흡수.

이렇게 하면 현재 회귀가 통계적으로 "끼워맞추던" 온도·위치·τ 효과의 상당 부분이 **물리식으로 사전 제거**되어, 회귀계수의 물리 해석력과 소표본 안정성이 동시에 좋아진다. 검증은 CLAUDE.md v3 기준 그대로 **SDM_corrected ↔ dOCV 연속상관(특히 양품만)** 으로 비교하면 된다.

### 도입 전 반드시 확인할 전제

- **Eact·TCV·Rser·Ceff 값**: A-1/A-2/B/C 물리 보정은 이 상수들이 필요. 보유 논문값(Eact=0.94 eV, TCV–SOC 곡선)을 초기값으로 쓰되, **여러 온도에서 측정한 자체 SDM 데이터로 캘리브레이션**해야 정밀.
- **경계성(미세) 불량 데이터**: CLAUDE.md가 강조한 대로, 현재 z=33짜리 명백한 단락만으로는 어떤 보정법이 우월한지 판가름 불가. 미세불량 데이터 확보 후 **보정법별 SDM↔dOCV 분리도**로 최종 선택해야 함.
